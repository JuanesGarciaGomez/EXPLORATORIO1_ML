Financial Transaction Fraud Dataset¶

Contexto¶

El fraude financiero es uno de los grandes problematicas de los sistemas bancarios en la actualidad. Con la digitalización de los pagos, las transacciones electrónicas crecen a un ritmo exponencial, el cual abre ventanas de oportunidad para actividades ilícitas que generan pérdidas millonarias tanto a instituciones como a usuarios.

Planteamiento del Problema¶

El fraude financiero representa una amenaza creciente para las instituciones bancarias, produciendo pérdidas millonarias. Debido a esto , es necesario aplicar técnicas de Machine Learning sobre el dataset para identificar de forma precisa y automatizada las transacciones fraudulentas dentro del entorno de datos.

Objetivo :¶

Explorar el dataset FraudShield para identificar y analizar la distribucion de variables entre las transacciones normales y fraudulentas para asociar patrones ante posibles fraudes.

Este notebook contiene el análisis exploratorio de datos de un dataset de transacciones financieras con el objetivo de entender patrones asociados al fraude.

fraud_by_type = df.groupby("Card_Type")["Fraud_Label"].apply(

).reset_index()
fraud_by_type.columns = ["Card_Type", "Fraud_Percentage"]

plt.figure(figsize=(8, 5))
bars = plt.bar(fraud_by_type["Card_Type"], fraud_by_type["Fraud_Percentage"], color=["#2E75B6", "#1F4E79", "#4BACC6"])

for bar, val in zip(bars, fraud_by_type["Fraud_Percentage"]):
    plt.text(bar.get_x() + bar.get_width()/2, bar.get_height() + 0.1, f"{val:.1f}%", ha="center", fontsize=11)

plt.title("Porcentaje de Fraude por Tipo de tarjeta", fontsize=13)
plt.xlabel("Tipo de tarjeta")
plt.ylabel("% Fraude")
plt.tight_layout()
